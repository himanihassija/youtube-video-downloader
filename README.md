# YouTube Video Downloader

A Python-based YouTube video downloader that allows users to download videos easily by providing a YouTube URL. The project focuses on simplicity and functionality, making it a beginner-friendly yet practical tool.

## Features
- Download YouTube videos using a URL
- Supports multiple video resolutions (based on availability)
- Simple and easy-to-use command-line interface
- Fast and lightweight

## Technologies Used
- Python
- pytube

## Installation
1. Clone the repository:
   git clone https://github.com/your-username/yt-video-downloader.git

2. Navigate to the project directory:
   cd yt-video-downloader

3. Install the required dependencies:
   pip install pytube

## Demo
https://github.com/himanihassija/youtube-video-downloader/blob/main/Yt%20video%20downloader.png


## Usage
1. Run the Python script:
   python downloader.py

2. Enter the YouTube video URL when prompted.

3. Choose the desired resolution if options are available.

4. The video will be downloaded to your local system.

## Project Purpose
This project was built to practice working with external Python libraries, handling user input, and creating a real-world utility application.
